Instalujemy express, ejs, morgan komendami
> npm install express
> npm install ejs
> npm install morgan
Startujemy stronę komendą 
> node index.js
Ewentualnie jeśli chcemy dane testowe od razu na stronie to można to wystartować serwer w taki sposób:
> POPULATE_DB=1 node index.js
I strona jest gotowa do użytku!
